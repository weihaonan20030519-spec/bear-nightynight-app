# Chuanqu/传趣
简体中文 [繁体中文](readme-tc.md)

一款开源中文字体基于Stick、游趣体字体衍生、修改，字形偏向于旧字形，风格俏皮、可爱。
## 简介
stick字体因笔形像笔杆一样以直线设计，而被视为一款既可爱调皮，亦有田园写意风格的字体。因为原版字体Stick支持字形较少，不能满足日常中文环境使用，因此简体中文版[「游趣体」](https://github.com/Des-Magmeta/ChildFunSans)应运而生，但本人萌生了旧字形化的念头，因此本人开坑了这个项目，并为其补充繁体字，改传承字形。

## 字符集
- [x] 繁体常用4808
- [x] big5一级字
- [x] GB2312、《通用规范汉字表》简繁体
- [x] 国际表意文字核心（iicore字符）
- [ ] 按需添加一些来自[「YShi笔书」](https://github.com/Steve-Yuu/YshiPen-Shuti)、霞鹜文楷TC的字符

## 注意事项
 1. 本人并非专业字体设计师，技术有限，难免会有疏漏
 2. 本字体参考YShi笔书Cl设计，主要区别在于「艹」断开，「禸」不穿、「永」首笔作横
 3. 只有标点居中版本
